Thermostat

Requirements:
- Mobile device with Android OS 4.0 or higher
- Minimum screen size: 3.7"

How to install:
- You should to allow to install applications from unknown source
- Copy Thermostat.apk to your device in any convenient way
- Run Thermostat.apk and click OK/Install when setup wizard ask you about security statements

How to run:
- Find Thermostat in list of your device applications
- Click on this icon
- Enjoy this
